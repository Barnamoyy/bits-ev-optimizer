# Changelog

## Latest Updates

### Removed 3D View
- Completely removed Three.js 3D terrain visualization
- Deleted TerrainView3D component
- Removed view mode toggle buttons

### Added Comprehensive Elevation Report
- New ElevationReport component with detailed analysis
- Shows elevation gain, loss, range, and average gradient
- Displays elevation statistics (max, min, start, end, net change)
- Gradient analysis with difficulty ratings
- Energy impact summary showing uphill energy, recovered energy, and net cost
- Driving recommendations based on terrain
- Terrain type classification (flat, rolling hills, hilly, mountainous)

### Removed All Emojis
- Dashboard: Removed all emoji icons, using only Lucide icons
- RouteSelector: Removed emoji decorations from dropdowns and buttons
- ElevationProfile: Removed emoji icons from tooltips and stat cards
- ElevationReport: Clean professional look without emojis

### Improved Layout
- Map now takes full width of right section
- Elevation Report displayed prominently below map
- Elevation Profile chart shown at bottom
- Better information hierarchy and spacing
